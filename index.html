Tag h1 á h6

Voltar

As tags HTML h1 a h6 são utilizadas para criar títulos em uma página web, com h1 sendo o título mais importante e h6 o menos importante. Elas indicam a hierarquia e a importância do texto dentro do conteúdo da página para os motores de busca e para a acessibilidade.

Tag P

Voltar

A tag p em HTML é usada para criar parágrafos de texto em uma página web. Ela define um bloco de texto que é separado dos parágrafos adjacentes por espaços verticais, tornando o conteúdo mais organizado e fácil de ler para os usuários.

Tag Mark

Voltar

A tag mark em HTML é usada para destacar parte do texto dentro de um parágrafo ou de qualquer elemento de texto. Quando você envolve um pedaço de texto com a tag mark, o texto dentro dela é realçado com um fundo amarelo por padrão, indicando visualmente que é importante ou relevante no contexto da página. Essa tag é especialmente útil para destacar palavras-chave ou termos de busca em resultados de pesquisa ou em trechos de texto específicos.

Tag Small

Voltar

A tag small em HTML é usada para definir texto ou conteúdo que é de menor importância ou tamanho em relação ao texto circundante. Geralmente, o navegador renderiza o texto dentro da tag small em um tamanho de fonte menor do que o texto padrão. Ela é comumente usada para indicar notas de rodapé, legendas, avisos legais ou qualquer outra informação secundária que seja relevante, mas não crucial para o entendimento principal do conteúdo da página. O uso da tag small é uma prática para melhorar a semântica e a acessibilidade do conteúdo web.

Tag I

Voltar

A tag i em HTML é uma tag de formatação que costumava ser usada para indicar que o texto deve ser exibido em itálico. No entanto, a partir do HTML5, a tag i não tem mais um significado semântico específico. Em vez disso, é recomendado usar a tag em para enfatizar o texto, indicando que ele tem importância contextualmente relevante, e a tag i é muitas vezes utilizada apenas para fins de estilização, como em casos de CSS personalizado para mudar a aparência visual do texto. É importante notar que, em HTML5, é preferível usar tags semânticas que expressem o significado do conteúdo, em vez de depender apenas de estilos visuais.

Tag U

Voltar

Para enfatizar ou destacar texto em HTML, é preferível usar outras tags semânticas, como strong para indicar importância forte ou em para indicar ênfase. Se você precisa de sublinhado para links, normalmente os navegadores já aplicam o sublinhado automaticamente aos links a sem a necessidade da tag u. Portanto, o uso da tag u não é uma prática recomendada em HTML5 e versões mais recentes.

Tag Strong

Voltar

A tag strong em HTML é usada para indicar que o texto contido dentro dela é semanticamente importante ou enfatizado. O texto envolto pela tag strong é geralmente exibido em negrito pelos navegadores da web padrão, embora o propósito principal da tag seja indicar a importância do conteúdo, não apenas seu estilo visual.

Tag Ol

Voltar

A tag ol em HTML é usada para criar listas ordenadas, onde os itens da lista são numerados automaticamente. "OL" é uma abreviação de "Ordered List" (Lista Ordenada). Cada item da lista é definido pela tag li, que fica dentro da tag ol. Os itens da lista são numerados sequencialmente por padrão, mas você pode personalizar o estilo de numeração usando CSS, se desejar.

Tag Ul

Voltar

A tag ul em HTML é usada para criar listas não ordenadas, onde os itens da lista não têm uma ordem específica e são marcados com símbolos, como pontos ou traços. "UL" é uma abreviação de "Unordered List" (Lista Não Ordenada). Cada item da lista é definido pela tag li, que fica dentro da tag ul. As listas não ordenadas são úteis quando a ordem dos itens não é importante, como em listas de itens ou características.

Tag Li

Voltar

A tag li em HTML é usada para criar itens em listas, sejam elas ordenadas (ol) ou não ordenadas (ul). LI é uma abreviação de "List Item" (Item de Lista). Cada item da lista é definido pela tag li, indicando um elemento na lista, seja numerado em listas ordenadas ou marcado em listas não ordenadas.

Tag A

Voltar

A tag a em HTML é usada para criar links de hipertexto. "A" é uma abreviação de "anchor" (âncora). É usada para criar links para outras páginas da web, arquivos, endereços de email ou qualquer outro URL. O atributo href dentro da tag especifica o destino do link.

Tag Hr

Voltar

A tag hr em HTML é usada para criar uma linha horizontal, representando uma divisão temática ou mudança de contexto em uma página da web. "HR" é uma abreviação de "horizontal rule" (linha horizontal). É frequentemente usada para separar conteúdos visuais, como seções diferentes de um artigo ou tópicos em uma página. A tag hr cria uma linha horizontal que se estende pela largura do contêiner pai, ajudando a organizar o conteúdo visualmente.

Tag Sub

Voltar

A tag sub em HTML é usada para definir texto subscrito, que aparece menor e abaixo da linha de base normal. É comumente usado para representar fórmulas químicas, notações matemáticas, datas e outras informações onde um texto menor deve aparecer abaixo da linha principal do texto.

Tag Sup

Voltar

A tag sup em HTML é usada para criar texto sobrescrito, que aparece menor e acima da linha de base normal. É comumente usado para representar expoentes em expressões matemáticas, unidades em fórmulas químicas ou notas de rodapé. O texto dentro da tag sup é exibido ligeiramente acima do texto normal, indicando que é um texto sobrescrito.

Tag Blockquote

Voltar

A tag blockquote em HTML é usada para indicar um bloco de texto que é uma citação de outra fonte. É frequentemente usado para citar longos trechos de texto de outros autores ou fontes, formatando-os como um bloco separado, muitas vezes com um recuo à esquerda. O elemento blockquote ajuda a diferenciar visualmente as citações do restante do conteúdo da página.
